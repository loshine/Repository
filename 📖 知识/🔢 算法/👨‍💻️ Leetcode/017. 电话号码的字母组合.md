# 电话号码的字母组合

## 题目

给定一个仅包含数字 2-9 的字符串，返回所有它能表示的字母组合。

给出数字到字母的映射如下（与电话按键相同）。注意 1 不对应任何字母。

![phone](https://i.imgur.com/3uYluI3.png)

## 解题

递归

```kotlin
class Solution {
    fun letterCombinations(digits: String): List<String> {
        val dict = arrayOf(" ", "", "abc", "def", "ghi", "jkl", "mno", "pqrs", "tuv", "wxyz")
        val results = mutableListOf<String>()

        return when {
            digits.isEmpty() -> emptyList()
            digits.length == 1 -> {
                dict[digits.toInt()].toCharArray().mapTo(results) { it.toString() }
                results
            }
            else -> {
                val list = letterCombinations(digits.substring(1, digits.length))
                val sb = StringBuilder()

                for (c in dict[digits.substring(0, 1).toInt()].toCharArray()) {
                    for (s in list) {
                        sb.append(c.toString()).append(s)
                        results.add(sb.toString())
                        sb.setLength(0)
                    }
                }
                results
            }
        }
    }
}
```
