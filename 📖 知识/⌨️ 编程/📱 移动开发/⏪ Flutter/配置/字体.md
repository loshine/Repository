# 字体

字体需要在`pubspec.yaml`中定义 fontFamily

```yaml
  # 像下面这样定义字体文件。每组字体文件都需要一个Family 
  # example:
  # fonts:
  # - family: Schyler
  # fonts:
  # - asset: fonts/Schyler-Regular.ttf
  # - asset: fonts/Schyler-Italic.ttf
  # style: italic
  # - family: Trajan Pro
  # fonts:
  # - asset: fonts/TrajanPro.ttf
  # - asset: fonts/TrajanPro_Bold.ttf
  # weight: 700
```
